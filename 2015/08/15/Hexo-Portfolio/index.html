Hexo Portfolio

Project Information

The portfolio theme for Hexo is exactly that, a portfolio theme for the Hexo CMS platform. Hexo is a node.js powered static site generator, which gives several benefits over using a standard Wordpress installation. The only pitfall was the lack of functionality for portfolios. To that end, the theme was created and is aimed specifically towards displaying existing work.

Web Components

The theme was developed using brackets with Node Package Manager, it is built using the Jade Templating Engine, SCSS, and skeleton.css. Due to the nature of it being statically generated, it can be hosted using GitHub pages.
