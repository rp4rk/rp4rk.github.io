NavSync

NavSync is an anchoring plugin for jQuery, built with both ease of use and performance in mind.

Project Information

NavSync is a jQuery plugin built to fill a niche in the jQuery plugin environment, there are plugins which can create waypoints and plugins which can manage scrolling to the selected element. There were no plugins which dealt with both these tasks without being part of a large and bulky framework, such as Bootstrap ScrollSpy. For lightweight websites solutions such as ScrollSpy were inappropriate. NavSync allows for an easy way to manage page waypoints through markup, and does not come as part of a bulky framework.

Project Goals

NavSync aims to provide:

  • A lightweight solution for navigation syncing, and do this in a manner which degrades gracefully for individuals with Javascript disabled.

  • Performance, NavSync reaches at least 60 fps with appropriate use of styling.

  • Ease of use, NavSync accomplishes everything it does with 1 line of code from the user.

  • Adaptability, NavSync only asks for semantic markup and all classes used are entirely configurable.

  • Customisability, NavSync does not enforce a style to the elements it modifies.

Features & Performance

NavSync runs at 60 frames per second, performance was a cornerstone of the project and several optimizations have been made in the code to reduce the amount of calls made.





This performance is delivered in a file which is < 1kb in size minified and gzipped, costing minimal amounts of bandwidth for mobile users. This is around ten times as small as the Bootstrap ScrollSpy alternative.
